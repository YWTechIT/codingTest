## math_function

1. math(): 수학계산을 요구하는 기능

```python
import math

# factorial(x)
print(math.factorial(5))
👉🏽 120
# sqrt(x): x의 제곱근
print(int(math.sqrt(49)))
👉🏽 7

# gcd(x): 최대 공약수
print(math.gcd(21, 14))
👉🏽 7

# 파이(pi) 출력
print(math.pi)
👉🏽 3.141592653589793

# 자연상수(e) 출력
print(math.e)
👉🏽 2.718281828459045
```
